# 声呐系统中波束形成的MATLAB实现

## 资源描述

本资源文件提供了一个用于声呐系统中波束形成的MATLAB实现程序。该程序主要用于仿真均匀线阵（ULA）CBF（常规波束形成）的波束图和方位估计（Direction of Arrival, DOA）空间谱。

## 功能介绍

1. **波束图仿真**：通过MATLAB代码生成均匀线阵（ULA）的波束图，展示不同方向上的波束响应。
2. **方位估计（DOA）空间谱**：计算并绘制方位估计的空间谱，帮助分析信号的到达方向。
